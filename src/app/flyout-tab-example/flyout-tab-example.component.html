Introduction

The flyout panel component displays a panel on the bottom-right side of the browser window. Initially the panel is closed. When the user clicks the panel tab, the panel animates open. When user clicks the panel tab again, the panel animates closed.

Technicals

Panel animation is done using css. The open class animates the panel open in 1 second. Similarly, when the open class is removed, the default css animates the panel closed in 1 second.

The flyout component takes two inputs:

  • A tabText property binding set to the desired tab text.
  • An ng-content input which passes all html content placed between the ancb-flyout-panel directive. The content is then dropped into the panel content section.

The flyout component emits a panelOpenChange event when the panel tab is clicked. The example hooks into, this event using the OnChanges lifecycle hook and changes the optional reanimate bound boolean property. Thus, when the panel is opened the meter is animated. Note the meter does not animate on close.
