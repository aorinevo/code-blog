Introduction

The meter component displays an svg meter. The radius, color, and stroke-width (thickness of the meter) can be changed.

Technicals

Meter animation leverages jquery but not to direclty update elements in the DOM. Instead, jquery takes an object containing the part and angle properties of the meter object. The call back function of the animate method modifies the appropriate svg properties (i.e. property d) that are bound to the meter component.

The meter component takes one input; an object of type Meter.
