Steps as of 7/13/2016
  1. Install iterm2 and set preferences. Download: https://www.iterm2.com/downloads.html (download the stable release) With admin access, move app into your Applications folder
  2. Change default shell to zsh over bash and set theme (I use Dark Background).
    • Change the default shell to zsh: 
            
chsh -s /bin/zsh
    • Set theme to Dark Background: iTerm -> preferences -> profiles -> colors -> load presets
  3. Clone and install Prezto.
    • Copy and paste the following bash command: 
      git clone --recursive https://github.com/sorin-ionescu/prezto.git "${{'{'}}ZDOTDIR:-$HOME{{'}'}}/.zprezto" 
  setopt EXTENDED_GLOB
    for rcfile in "${{'{'}}ZDOTDIR:-$HOME{{'}'}}"/.zprezto/runcoms/^README.md(.N); do
      ln -s "$rcfile" "${{'{'}}ZDOTDIR:-$HOME{{'}'}}/.${{'{'}}rcfile:t{{'}'}}"
    done
  4. Install your favorite IDE (I use Atom).
    • Download dmg from https://atom.io/
    • With admin access, move app into your Applications folder
  5. Modify ~/.zshrc file by copying the following lines into the end of the file. 
    export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.7.0_80.jdk/Contents/Home
export MAVEN_HOME=/usr/local/Cellar/maven/3.3.9/libexec
export PATH=$MAVEN_HOME/bin:$JAVA_HOME/bin:$PATH
export MAVEN_OPTS="-Xmx1024m -Xms1024m -XX:MaxPermSize=1162m"
export M2_OPTS=$MAVEN_OPTS
export PATH=/Users/u060014/texlive/2016/bin/x86_64-darwin:$PATH

# Source Prezto.
if [[ -s "${{'{'}}ZDOTDIR:-$HOME{{'}'}}/.zprezto/init.zsh" ]]; then
  source "${{'{'}}ZDOTDIR:-$HOME{{'}'}}/.zprezto/init.zsh"
fi
  6. Change the theme in ~/.zpreztorc to steeef (Line 95, at time of writing this doc)
Notes:
  • If you are using sublime, make sure to add an alias in ~/.zshrc file: 
    #Applications
alias sublime="open -a /Applications/Sublime.app"
  • Also for sublime users, make sure you have sublime in your Applications folder and name the app Sublime.app (The default name is Sublime Text.app)
